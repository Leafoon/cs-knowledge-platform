## D.6 开源项目与模板

### 官方模板（可直接使用）

访问：https://python.langchain.com/docs/templates

**热门模板：**

| 模板名 | 说明 |
|--------|------|
| `rag-chroma` | 基于 Chroma 的 RAG |
| `rag-pinecone` | 基于 Pinecone 的 RAG |
| `rag-conversation` | 对话式 RAG |
| `openai-functions-agent` | OpenAI Function Calling Agent |
| `research-assistant` | 研究助手（多步骤 Agent） |
| `extraction-anthropic` | 信息提取（Anthropic） |
| `sql-pgvector` | SQL + 向量数据库混合查询 |

**使用方法：**

```bash
# 安装 LangChain CLI
pip install langchain-cli

# 从模板创建项目
langchain app new my-app --package rag-chroma

# 运行
cd my-app
langchain serve
```
