# 第九章 算术逻辑单元（ALU）

ALU（Arithmetic Logic Unit）是CPU的核心运算部件，负责执行所有的算术和逻辑运算。本章讲解ALU的功能、结构、标志位产生，以及经典的74181 ALU芯片。

---

## 9.1 ALU的功能与结构

### 9.1.1 ALU的基本功能

<div data-component="ALUBlockDiagram"></div>

ALU接收两个操作数和一个操作码，输出运算结果和状态标志。

**基本运算类型**：

| 运算类型 | 具体操作 | 说明 |
|----------|---------|------|
| 算术运算 | 加、减、乘、除、加1、减1 | 数值计算 |
| 逻辑运算 | 与、或、非、异或、同或 | 位运算 |
| 移位运算 | 左移、右移（逻辑/算术/循环） | 数据移动 |
| 比较运算 | 大于、小于、等于 | 条件判断 |

### 9.1.2 ALU的输入输出

```
┌─────────────────────────────────────┐
│              ALU                    │
│                                     │
│  A(n位) ──→│               │──→ F(n位)  结果
│  B(n位) ──→│   运算逻辑     │──→ ZF      零标志
│  Op(m位) ─→│               │──→ SF      符号标志
│  Cin ─────→│               │──→ OF      溢出标志
│                                     │──→ CF      进位标志
└─────────────────────────────────────┘
```

| 信号 | 方向 | 说明 |
|------|------|------|
| A | 输入 | 操作数A |
| B | 输入 | 操作数B |
| Op | 输入 | 操作码（选择运算类型） |
| Cin | 输入 | 进位输入 |
| F | 输出 | 运算结果 |
| ZF | 输出 | 零标志 |
| SF | 输出 | 符号标志 |
| OF | 输出 | 溢出标志 |
| CF | 输出 | 进位标志 |

## 9.3 标志位的产生

<div data-component="FlagRegisterViz"></div>

### 9.3.1 ZF（零标志）

**定义**：运算结果为0时，ZF=1。

**产生逻辑**：

$$
ZF = \overline{F_{n-1} + F_{n-2} + \cdots + F_1 + F_0}
$$

即结果所有位的**或非**（NOR）。

**硬件实现**：一个多输入NOR门。

### 9.3.2 SF（符号标志）

**定义**：运算结果的最高位（符号位）。

$$
SF = F_{n-1}
$$

**含义**：SF=1表示结果为负数（补码表示）。

### 9.3.3 OF（溢出标志）

**定义**：有符号数运算溢出时，OF=1。

**产生方法**：

**方法一：进位法**

$$
OF = C_{n-1} \oplus C_n
$$

其中$C_{n-1}$是最高数值位的进位，$C_n$是符号位的进位。

**方法二：双符号位法**

$$
OF = S_{s1} \oplus S_{s2}
$$

**方法三：同号相加判断法**

$$
OF = A_s \cdot B_s \cdot \overline{F_s} + \overline{A_s} \cdot \overline{B_s} \cdot F_s
$$

### 9.3.4 CF（进位标志）

**定义**：无符号数运算产生进位/借位时，CF=1。

**加法**：$CF = C_n$（最高位进位输出）

**减法**：$CF = \overline{C_n}$（借位时最高位进位为0）

### 9.3.5 标志位汇总

| 标志 | 名称 | 产生条件 | 用途 |
|------|------|---------|------|
| ZF | 零标志 | 结果为0 | 判断是否相等 |
| SF | 符号标志 | 结果为负 | 判断正负 |
| OF | 溢出标志 | 有符号溢出 | 判断溢出 |
| CF | 进位标志 | 无符号进位 | 判断进位 |

### 9.3.6 标志位的应用

**条件跳转指令**根据标志位决定是否跳转：

| 指令 | 条件 | 含义 |
|------|------|------|
| JZ/JE | ZF=1 | 结果为零/相等 |
| JNZ/JNE | ZF=0 | 结果非零/不相等 |
| JS | SF=1 | 结果为负 |
| JNS | SF=0 | 结果为正 |
| JO | OF=1 | 溢出 |
| JNO | OF=0 | 无溢出 |
| JC/JB | CF=1 | 进位/低于 |
| JNC/JAE | CF=0 | 无进位/高于等于 |

## 9.4 74181 ALU芯片

<div data-component="SN74181Explorer"></div>

### 9.4.1 概述

74181是一种经典的**4位ALU**芯片，支持16种算术运算和16种逻辑运算。

### 9.4.2 引脚

| 引脚 | 方向 | 说明 |
|------|------|------|
| A3~A0 | 输入 | 操作数A |
| B3~B0 | 输入 | 操作数B |
| S3~S0 | 输入 | 功能选择 |
| M | 输入 | 模式控制（0=算术，1=逻辑） |
| Cn | 输入 | 进位输入 |
| F3~F0 | 输出 | 运算结果 |
| Cn+4 | 输出 | 进位输出 |
| A=B | 输出 | A等于B标志 |
| P, G | 输出 | 进位传播和生成 |

### 9.4.3 功能表

| S3S2S1S0 | 算术运算（M=0） | 逻辑运算（M=1） |
|-----------|----------------|----------------|
| 0000 | F = A | F = $\overline{A}$ |
| 0001 | F = A + B | F = $\overline{A+B}$ |
| 0010 | F = A + $\overline{B}$ | F = $\overline{A} \cdot B$ |
| 0011 | F = -1（全1） | F = 0 |
| 0100 | F = A + A$\overline{B}$ | F = $\overline{A \cdot B}$ |
| 0101 | F = (A+B) + A$\overline{B}$ | F = $\overline{B}$ |
| 0110 | F = A - B - 1 | F = A ⊕ B |
| 0111 | F = A$\overline{B}$ - 1 | F = A$\overline{B}$ |
| 1000 | F = A + AB | F = $\overline{A}$ + B |
| 1001 | F = A + B | F = $\overline{A \oplus B}$ |
| 1010 | F = (A+$\overline{B}$) + AB | F = B |
| 1011 | F = AB - 1 | F = AB |
| 1100 | F = A + A | F = 1 |
| 1101 | F = (A+B) + A | F = A + $\overline{B}$ |
| 1110 | F = (A+$\overline{B}$) + A | F = A + B |
| 1111 | F = A - 1 | F = A |

### 9.4.4 74181的进位逻辑

74181内部使用**超前进位**逻辑：

- **G（Generate）**：$G = A \cdot B$（产生进位）
- **P（Propagate）**：$P = A \oplus B$（传播进位）

$$
C_{n+1} = G + P \cdot C_n
$$

### 9.4.5 74181的级联

<div data-component="CarrySelectAdder"></div>

单片74181只能处理4位数据。对于更宽的数据（如16位、32位），需要多片级联。

**串行级联**：

```
[74181] → C4 → [74181] → C8 → [74181] → C12 → [74181] → C16
 片0              片1              片2              片3
```

- 简单但速度慢（进位串行传递）

**并行级联（使用74182超前进位发生器）**：

```
[74181] → G0,P0 ─→ ┌──────────┐
[74181] → G1,P1 ─→ │  74182   │ → C4, C8, C12, C16
[74181] → G2,P2 ─→ │ 超前进位  │
[74181] → G3,P3 ─→ └──────────┘
```

- 速度快（组间超前进位）

## 9.6 本章典型例题

### 例题1：74181功能分析

**题目**：用74181实现 $A + B$，需要设置什么控制信号？

**解答**：

查功能表，$A + B$ 对应 $S3S2S1S0 = 1001$，$M = 0$（算术模式），$C_{in} = 0$。

### 例题2：标志位计算

**题目**：8位ALU计算 $0x7F + 0x01$，求ZF、SF、OF、CF。

**解答**：

$0x7F + 0x01 = 0x80 = 10000000_2$

- ZF = 0（结果非零）
- SF = 1（最高位为1）
- OF = 1（$0x7F=127$，$0x01=1$，$127+1=128>127$，有符号溢出）
- CF = 0（无进位）

### 例题3：16位ALU设计

**题目**：用4片74181和1片74182设计一个16位ALU，分析其进位延迟。

**解答**：

- 4片74181并行连接，每片处理4位
- 74182接收4片的 $G, P$ 信号，直接产生 $C_4, C_8, C_{12}$
- 进位延迟：约2级门延迟（74182的延迟）+ 74181的内部延迟
- 总延迟远小于串行级联的 $4 \times$ 单片延迟

### 例题4：ALU功能扩展

**题目**：如何用ALU实现比较操作 $A < B$？

**解答**：

计算 $A - B$，检查标志位：
- 如果 $OF \neq SF$（有符号比较）：$A < B$
- 或者 $CF = 1$（无符号比较）：$A < B$

## 9.7 本章常见误区与易错点

1. **OF和CF的区别**：OF用于有符号数溢出判断，CF用于无符号数进位判断。

2. **ZF的产生**：ZF是结果所有位的或非，不是异或。

3. **SF就是结果的最高位**：不需要额外计算。

4. **74181的M信号**：M=0是算术模式，M=1是逻辑模式。

5. **超前进位的"超前"**：不是指时间上提前，而是指直接计算进位，不等待低位进位传递。

6. **ALU的位数**：ALU的位数决定了它一次能处理的数据宽度。

## 9.10 深入理解ALU的设计

### 9.10.1 ALU的基本结构

ALU是CPU的核心运算部件，负责执行所有的算术和逻辑运算。

**ALU的输入**：
- 操作数A（n位）
- 操作数B（n位）
- 操作码（m位）
- 进位输入（Cin）

**ALU的输出**：
- 运算结果F（n位）
- 进位输出（Cout）
- 零标志（ZF）
- 符号标志（SF）
- 溢出标志（OF）
- 进位标志（CF）

### 9.10.2 ALU的功能实现

ALU内部由多个功能部件组成：

```
┌──────────────────────────────────────┐
│                ALU                    │
│                                      │
│  ┌─────┐   ┌─────┐   ┌─────┐      │
│  │加法器│   │逻辑  │   │移位器│      │
│  │     │   │运算  │   │     │      │
│  └──┬──┘   └──┬──┘   └──┬──┘      │
│     │         │         │          │
│     └────┬────┘         │          │
│          ↓              ↓          │
│       ┌─────────────────┐         │
│       │    MUX          │         │
│       │  (结果选择)      │         │
│       └────────┬────────┘         │
│                ↓                   │
│           结果输出                  │
│                                      │
│  ┌───────────────────────────────┐  │
│  │        标志位产生逻辑          │  │
│  └───────────────────────────────┘  │
└──────────────────────────────────────┘
```

### 9.10.3 加法器在ALU中的作用

加法器是ALU中最重要的部件，不仅用于加法运算，还用于：
- 减法：$A - B = A + (-B) = A + \overline{B} + 1$
- 比较：$A - B$，检查标志位
- 地址计算：基址 + 偏移

### 9.10.4 逻辑运算的实现

逻辑运算是**按位**进行的，每一位独立计算：

| 运算 | 逻辑表达式 | 说明 |
|------|-----------|------|
| AND | $F_i = A_i \cdot B_i$ | 按位与 |
| OR | $F_i = A_i + B_i$ | 按位或 |
| XOR | $F_i = A_i \oplus B_i$ | 按位异或 |
| NOT | $F_i = \overline{A_i}$ | 按位取反 |

### 9.10.5 移位器的实现

移位器可以实现左移和右移操作：

| 操作 | 说明 | 用途 |
|------|------|------|
| 逻辑左移 | 高位移出，低位补0 | 乘以2的幂 |
| 逻辑右移 | 低位移出，高位补0 | 无符号除以2的幂 |
| 算术右移 | 低位移出，高位补符号位 | 有符号除以2的幂 |
| 循环移位 | 移出的位从另一端移入 | 循环缓冲 |

| 1101 | F = A+B̄ | |

### 9.11.2 74181的进位逻辑

74181内部使用超前进位逻辑：

**生成信号**：$G = A \cdot B$

**传播信号**：$P = A \oplus B$

**进位公式**：$C_{i+1} = G_i + P_i \cdot C_i$

74181输出组级别的G和P信号，可以连接74182超前进位发生器，实现组间超前进位。

### 9.11.3 74181+74182的级联

**16位ALU的设计**：

```
[74181] → G0,P0 ─→ ┌──────────┐
[74181] → G1,P1 ─→ │  74182   │ → C4, C8, C12
[74181] → G2,P2 ─→ │ 超前进位  │
[74181] → G3,P3 ─→ └──────────┘
```

74182接收4片74181的G、P信号，直接产生C4、C8、C12进位信号。

**性能**：
- 4位CLA延迟：4级门延迟
- 16位两级CLA延迟：6~7级门延迟
- 远小于串行级联的16级门延迟

## 9.12 标志位的深入分析

### 9.12.1 ZF（零标志）

**产生逻辑**：结果所有位的或非（NOR）。

$$
ZF = \overline{F_{n-1} + F_{n-2} + \cdots + F_1 + F_0}
$$

**硬件实现**：一个多输入NOR门。

**用途**：
- 判断运算结果是否为0
- 判断两个数是否相等（A-B的结果ZF=1表示A=B）

### 9.12.2 SF（符号标志）

**产生逻辑**：结果的最高位。

$$
SF = F_{n-1}
$$

**用途**：
- 判断运算结果的正负
- 有符号数比较

### 9.12.3 OF（溢出标志）

**产生逻辑**：最高数值位进位与符号位进位异或。

$$
OF = C_{n-1} \oplus C_n
$$

**用途**：
- 判断有符号数运算是否溢出
- 溢出时结果不可信

### 9.12.4 CF（进位标志）

**产生逻辑**：
- 加法：$CF = C_n$（最高位进位输出）
- 减法：$CF = \overline{C_n}$（借位时最高位进位为0）

**用途**：
- 判断无符号数运算是否溢出
- 无符号数比较

### 9.12.5 标志位的应用

**条件跳转指令**根据标志位决定是否跳转：

| 指令 | 条件 | 含义 |
|------|------|------|
| JZ/JE | ZF=1 | 结果为零/相等 |
| JNZ/JNE | ZF=0 | 结果非零/不相等 |
| JS | SF=1 | 结果为负 |
| JNS | SF=0 | 结果为正 |
| JO | OF=1 | 溢出 |
| JNO | OF=0 | 无溢出 |
| JC/JB | CF=1 | 进位/低于（无符号） |
| JNC/JAE | CF=0 | 无进位/高于等于（无符号） |
| JG/JNLE | ZF=0且SF=OF | 大于（有符号） |
| JLE/JNG | ZF=1或SF≠OF | 小于等于（有符号） |

## 9.14 本章综合练习

### 练习1：ALU功能分析

**题目**：用74181实现 $A + B$ 和 $A - B$，需要设置什么控制信号？

**解答**：

**$A + B$**：
- S3S2S1S0 = 1001（查功能表）
- M = 0（算术模式）
- Cin = 0

**$A - B$**：
- 方法1：S3S2S1S0 = 0110，M = 0，Cin = 1（$A - B - 1 + 1 = A - B$）
- 方法2：先求 $\overline{B}$（逻辑运算），再做 $A + \overline{B} + 1$

### 练习2：标志位计算

**题目**：8位ALU计算 $0x7F + 0x01$，求ZF、SF、OF、CF。

**解答**：

$0x7F + 0x01 = 0x80 = 10000000_2$

- ZF = 0（结果非零）
- SF = 1（最高位为1）
- OF = 1（$0x7F=127$，$0x01=1$，$127+1=128>127$，有符号溢出）
- CF = 0（无进位）

### 练习3：16位ALU设计

**题目**：用4片74181和1片74182设计一个16位ALU，分析其进位延迟。

**解答**：

- 4片74181并行连接，每片处理4位
- 74182接收4片的G、P信号，直接产生C4、C8、C12
- 进位延迟：
  - 第一级（74181内部）：2级门延迟（生成G、P）+ 2级门延迟（进位）
  - 第二级（74182）：2级门延迟
  - 总延迟：约6~7级门延迟

## 9.15 常见错误与易错点

1. **OF和CF的区别**：OF用于有符号数溢出判断，CF用于无符号数进位判断。

2. **ZF的产生**：ZF是结果所有位的或非，不是异或。

3. **SF就是结果的最高位**：不需要额外计算。

4. **74181的M信号**：M=0是算术模式，M=1是逻辑模式。

5. **超前进位的"超前"**：不是指时间上提前，而是指直接计算进位。

6. **ALU的位数**：ALU的位数决定了它一次能处理的数据宽度。

7. **74181的功能表**：需要仔细查表，不要混淆算术和逻辑运算。

### 练习4：ALU功能扩展

### 练习5：ALU性能分析

**题目**：某ALU使用4位74181芯片，16位数据宽度。如果每个74181的延迟是10ns，74182的延迟是5ns。求16位加法的总延迟。

**解答**：

使用两级超前进位：
- 第一级（74181内部）：10ns
- 第二级（74182）：5ns
- 总延迟：10 + 5 = 15ns

如果使用串行级联：$4 \times 10 = 40$ns

加速比：$40 / 15 = 2.67$

### 练习6：标志位综合

$0x80 + 0x80 = 0x100$

结果（8位）= $0x00$

- ZF = 1（结果为0）
- SF = 0（最高位为0）
- OF = 1（$-128 + (-128) = -256$，超出范围）

## 9.21 ALU设计中的常见错误

### 9.21.1 溢出判断错误

**错误**：只检查符号位就判断溢出。

**正确**：必须检查两个操作数的符号和结果的符号。

**【例 9-6】** 判断 $0x7F + 0x01$ 是否溢出。

- $0x7F = 127$（有符号），$0x01 = 1$
- 结果 = $0x80 = -128$（有符号）
- 两个正数相加得到负数 → **溢出！**

### 9.21.2 OF和CF混淆

**错误**：OF=1一定有溢出，CF=1一定有进位。

**正确**：
- OF用于有符号数溢出判断
- CF用于无符号数进位判断
- 同一个运算结果可能 OF=0 但 CF=1

**【例 9-7】** $0xFF + 0x01 = 0x100$

- 有符号：$-1 + 1 = 0$，**OF=0**
- 无符号：$255 + 1 = 256$，**CF=1**

### 9.21.3 74181功能表查错

**错误**：混淆算术运算和逻辑运算。

**正确**：M=0是算术模式，M=1是逻辑模式。

### 9.21.4 超前进位延迟计算错误

**错误**：超前进位延迟与位数成正比。

**正确**：超前进位延迟与位数无关（$O(1)$），但硬件复杂度与位数成正比。
