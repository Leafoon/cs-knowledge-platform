## 28.1 稀疏计算基础

### 28.1.1 为什么需要稀疏计算？

深度学习中的稀疏性来源于多种技术：

| 稀疏来源 | 典型稀疏度 | 说明 |
|----------|-----------|------|
| **权重剪枝** | 50%-95% | 移除不重要的权重 |
| **结构化剪枝** | 30%-80% | 移除整个通道/层 |
| **注意力稀疏** | 60%-90% | 稀疏注意力机制 |
| **激活稀疏** | 30%-70% | ReLU 产生的零值 |
| **MoE 模型** | 80%-95% | 每次只激活部分专家 |

**稀疏计算的加速潜力**：

设张量中非零元素比例为 $\rho$（稀疏度 = $1 - \rho$），理论上：

$$\text{加速比} \leq \frac{1}{\rho}$$

例如，$\rho = 0.1$（90% 稀疏）理论上可获得 10× 加速。

但实际加速受限于：
1. 稀疏格式的元数据开销
2. 不规则访存导致的缓存失效
3. 负载不均衡

### 28.1.2 稀疏张量的数学定义

一个 $m \times n$ 的稀疏矩阵 $A$ 中，大部分元素为零。非零元素集合定义为：

$$\text{nnz}(A) = \{(i, j) \mid A_{ij} \neq 0\}$$

稀疏矩阵的存储目标是以 $O(\text{nnz})$ 而非 $O(m \times n)$ 的空间存储矩阵。

```c
// CSR SpMM：A(m×k, sparse) @ B(k×n, dense) = C(m×n, dense)
void spmm_csr(
    const float* A_data,
    const int* A_indices,
    const int* A_indptr,
    const float* B,        // k×n，行优先
    float* C,              // m×n，行优先
    int m, int k, int n
) {
    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            C[i * n + j] = 0.0f;
        }
        for (int p = A_indptr[i]; p < A_indptr[i+1]; p++) {
            int col = A_indices[p];
            float val = A_data[p];
            // 向量化：一次处理 n 个元素
            for (int j = 0; j < n; j++) {
                C[i * n + j] += val * B[col * n + j];
            }
        }
    }
}
```

```python
import numpy as np

def magnitude_prune(weights, sparsity=0.9):
    """幅度剪枝：将最小的权重置零"""
    threshold = np.percentile(np.abs(weights), sparsity * 100)
    mask = np.abs(weights) >= threshold
    pruned_weights = weights * mask
    return pruned_weights, mask

def structured_prune(weights, sparsity=0.5, axis=0):
    """结构化剪枝：移除整个通道"""
    # 计算每个通道的 L1 范数
    norms = np.sum(np.abs(weights), axis=tuple(range(1, weights.ndim)))
    threshold = np.percentile(norms, sparsity * 100)
    channel_mask = norms >= threshold
    pruned_weights = weights[channel_mask]
    return pruned_weights, channel_mask
```

- **业务意义**：上面的片段不是孤立 API 演示，而是在说明本章主题如何落到真实模型部署流程中；阅读时应关注输入、输出、目标硬件和运行时边界，而不是只记住函数名。
- **TVM 内部机制**：该片段通常会触发前端导入、IR 构造、Pass 优化、Lowering、CodeGen 或 Runtime 调用中的一个或多个阶段；如果结果异常，应优先判断问题发生在哪一层 IR，而不是直接修改最终生成代码。
- **适用场景**：它适合用于教学验证、最小复现、性能基线建立或单一优化点确认；在生产环境中还需要补充版本锁定、输入分布校验、目标设备基准测试和回滚策略。
- **限制条件**：示例默认省略了异常处理、数据集规模、硬件差异、编译缓存和长期维护成本；当模型包含动态形状、自定义算子或厂商私有后端时，需要结合本章后续工程经验进行扩展。
```python
import tvm
from tvm import relay
import numpy as np

def create_sparse_bert_attention(seq_len=512, num_heads=12, head_dim=64,
                                  block_size=64, sparsity=0.8):
    """创建块稀疏 BERT 注意力层"""

    # 输入
    Q = relay.var("Q", shape=(1, num_heads, seq_len, head_dim))
    K = relay.var("K", shape=(1, num_heads, seq_len, head_dim))
    V = relay.var("V", shape=(1, num_heads, seq_len, head_dim))

    # 稀疏 mask（块级别的）
    num_blocks = seq_len // block_size
    mask = relay.var("mask", shape=(1, num_heads, num_blocks, num_blocks),
                     dtype="float32")

    # 注意力分数
    scale = relay.const(1.0 / np.sqrt(head_dim))
    scores = relay.op.nn.batch_matmul(Q, relay.op.transpose(K, [0, 1, 3, 2]))
    scores = relay.op.multiply(scores, scale)

    # 应用块稀疏 mask
    # 将 mask 扩展到元素级别
    mask_expanded = relay.op.repeat(mask, block_size, axis=2)
    mask_expanded = relay.op.repeat(mask_expanded, block_size, axis=3)
    scores = relay.op.where(
        mask_expanded > relay.const(0.0),
        scores,
        relay.const(-1e9, dtype="float32")
    )

    # Softmax
    attn_weights = relay.op.nn.softmax(scores, axis=-1)

    # 加权求和
    output = relay.op.nn.batch_matmul(attn_weights, V)

    # 输出投影
    W_out = relay.var("W_out", shape=(num_heads * head_dim, num_heads * head_dim))
    output_flat = relay.op.reshape(output, (1, seq_len, num_heads * head_dim))
    projected = relay.op.nn.dense(output_flat, W_out)

    func = relay.Function([Q, K, V, mask, W_out], projected)
    return tvm.IRModule.from_expr(func)
```

### 28.99.2 业务意义

1. 稀疏计算 的业务价值不只是让模型跑得更快，而是让同一个模型可以在不同成本、功耗和延迟约束下交付。
2. 在服务器场景中，核心指标通常是吞吐、P95/P99 延迟、资源利用率和多租户隔离。
3. 在移动端场景中，核心指标通常是首帧时间、持续发热、内存峰值和包体大小。
4. 在嵌入式场景中，核心指标通常是 Flash 占用、静态内存、实时性和掉电恢复能力。
5. 在云端批处理场景中，编译时间可以接受更长，但调优记录和缓存复用变得非常重要。
6. 在在线服务场景中，编译产物需要可回滚、可审计、可灰度，而不能只在开发机上验证。
7. 业务方关心的是 SLA、成本和稳定性，编译器工程师关心的是 IR 正确性、优化空间和后端能力。
8. 优秀的 TVM 项目需要把这两类语言翻译成共同的指标体系。
9. 当优化收益只有少量百分点时，应评估它是否值得引入新的维护复杂度。

### 28.99.3 TVM 内部机制

1. TVM 的核心机制是分层表示和逐层降低，高层 IR 保留模型语义，低层 IR 暴露循环、内存和并行结构。
2. Relay 更适合表达静态计算图和传统深度学习算子。
3. Relax 更强调动态形状、函数式组合和跨层优化。
4. TIR 是最终性能调优的关键，因为它决定循环嵌套、内存作用域、向量化和线程映射。
5. PassContext 会影响优化级别、禁用或启用的 pass，以及某些后端特定配置。
6. Target 不只是字符串，它包含硬件特征、指令集、运行时约定和 codegen 选择。
7. Executor 决定模型入口如何组织，Runtime 决定编译产物如何加载和调用。
8. 参数绑定会影响常量折叠、内存规划和代码体积。
9. Layout rewrite 会影响算子融合和后端库调用。
10. AutoTVM 与 MetaSchedule 的调优记录会把搜索结果沉淀为可复用资产。

### 28.99.4 适用场景

1. 当模型结构相对稳定、目标硬件明确、性能收益可以通过基准测试确认时，稀疏计算 相关技术最容易发挥价值。

3. 当模型中存在框架运行时开销、算子融合机会或布局转换冗余时，编译优化通常能带来明显收益。

### 28.99.5 限制条件

1. TVM 并不能自动解决所有性能问题，尤其不能替代对模型结构、硬件层次和数据分布的理解。
2. 如果前端导入阶段已经丢失语义，后续 pass 很难恢复原始意图。
3. 如果目标硬件缺少成熟 codegen 或 runtime 支持，理论上的 IR 优化收益可能无法兑现。

5. 如果输入分布与校准集或 benchmark 数据差异较大，性能和精度结论都可能失真。
6. 如果编译参数没有固定，调优结果难以复现。
7. 如果只看平均延迟，可能忽略 P99 抖动和内存峰值。
8. 如果只在开发机验证，可能忽略生产驱动、内核版本和设备温控策略。
9. 如果项目缺少 IR dump 和 artifact 存档，出现线上问题时很难追溯。

### 28.99.9 与同类系统对比

1. 与 TensorRT 相比，TVM 的硬件覆盖更开放，TensorRT 在 NVIDIA GPU 上的工程成熟度更高。

3. 与 MLIR 相比，TVM 更像面向深度学习部署的完整编译器，MLIR 更像可构建编译器的基础设施。
4. 与 ONNX Runtime 相比，TVM 更关注提前编译和内核生成，ONNX Runtime 更强调运行时图优化和 execution provider 生态。
5. 与 Triton 相比，TVM 覆盖端到端模型编译，Triton 更适合手写或自动生成 GPU kernel。
6. 与 TFLite 相比，TVM 的后端扩展更灵活，TFLite 在移动端生态和模型格式上更标准化。
7. 与厂商 NPU SDK 相比，TVM 更中立，厂商 SDK 往往能访问更底层的私有能力。

### 28.11.1 ELL（ELLPACK）格式

ELL 格式将稀疏矩阵存储为密集数组，每行填充到相同长度：

```
[ 0  0  3  0 ]
[ 0  4  0  0 ]
[ 5  0  0  6 ]

ELL 格式（max_nnz_per_row = 2）：
values  = [[1, 2],    每行的非零值（填充到 max_nnz）
```

```python
    """稀疏算子的代价模型"""

    # 1. 计算量估计
    flops = matrix_stats["nnz"] * schedule["vectorize_width"]

    # 2. 内存访问估计
    # 稀疏访问的缓存命中率更低
    cache_hit_rate = estimate_sparse_cache_hit_rate(
        schedule, matrix_stats)

    load_balance = estimate_load_balance(
        schedule, matrix_stats["row_nnz_distribution"])

    # 4. 综合代价
    compute_cost = flops / schedule["compute_throughput"]
    memory_cost = matrix_stats["nnz"] * 4 / schedule["memory_bandwidth"]
    imbalance_cost = (1.0 - load_balance) * compute_cost

    total_cost = max(compute_cost, memory_cost) + imbalance_cost
```

```python
    """稀疏矩阵的分块规则"""

    def apply(self, sch, block):
        # 获取矩阵的稀疏统计信息
        sparse_stats = get_sparse_stats(block)

        # 根据稀疏度选择分块策略
        if sparse_stats["sparsity"] > 0.9:

        # 应用分块

        io, ii = sch[block].split(i, factor=tile_size)
        jo, ji = sch[block].split(j, factor=tile_size)
        ko, ki = sch[block].split(k, factor=tile_size)

        sch[block].reorder(io, jo, ko, ii, ki, ji)

        return sch
```

```python
    nnz = np.count_nonzero(weights)
    total = weights.size
    sparsity = 1.0 - nnz / total

    if sparsity < sparsity_threshold:
        # 稀疏度不够，保持密集格式
        return weights, "dense"

    # 选择最优的稀疏格式
    if is_block_sparse(weights):
        # 块稀疏 → BSR
        bsr = to_bsr(weights, block_size=(16, 16))
        return bsr, "bsr"
    elif is_diagonal_dominant(weights):
        # 对角占优 → CSR（保留对角线附近的元素）
        csr = to_csr(weights)
        return csr, "csr"
    else:
        # 一般稀疏 → CSR
        csr = to_csr(weights)
        return csr, "csr"
```

```python
    """Flash Attention 的稀疏版本"""

    batch, heads, seq_len, dim = Q.shape
    num_blocks = seq_len // block_size

    # 初始化输出
    O = torch.zeros_like(Q)

    for i in range(num_blocks):
        # 加载 Q 的一个块
        q_block = Q[:, :, i*block_size:(i+1)*block_size, :]

        # 初始化局部统计量
        m_i = torch.full((batch, heads, block_size, 1), float('-inf'))
        l_i = torch.zeros((batch, heads, block_size, 1))

        for j in range(num_blocks):
            # 检查稀疏 mask
            if sparsity_mask[i, j] == 0:
                continue  # 跳过被 mask 的块

            # 加载 K, V 的一个块
            k_block = K[:, :, j*block_size:(j+1)*block_size, :]
            v_block = V[:, :, j*block_size:(j+1)*block_size, :]

            # 计算注意力分数
            scores = torch.matmul(q_block, k_block.transpose(-2, -1))
            scores = scores / (dim ** 0.5)

            # 更新统计量
            m_new = torch.max(m_i, scores.max(dim=-1, keepdim=True).values)
            exp_scores = torch.exp(scores - m_new)

            # 更新输出
            l_new = torch.exp(m_i - m_new) * l_i + exp_scores.sum(dim=-1, keepdim=True)
            O[:, :, i*block_size:(i+1)*block_size, :] = (
                torch.exp(m_i - m_new) * l_i * O[:, :, i*block_size:(i+1)*block_size, :] +
                torch.matmul(exp_scores, v_block)
            ) / l_new

            m_i = m_new
            l_i = l_new

    return O
```

```python
    """稀疏混合专家层"""

    # 1. 路由：选择 top-k 个专家
    router_logits = router(input)
    top_k_indices = torch.topk(router_logits, k=top_k, dim=-1).indices
    top_k_weights = torch.softmax(
        torch.gather(router_logits, -1, top_k_indices), dim=-1)

        selected_tokens = input[mask]

        # 在专家上计算
        expert_output = expert_weights[expert_idx[mask]](selected_tokens)

        # 累加到输出
        output[mask] += weight[mask] * expert_output

    return output
```

```python
    """稀疏算子的 Roofline 分析"""

    # 硬件参数
    peak_compute = hardware_spec["peak_flops"]  # FLOPS
    peak_bandwidth = hardware_spec["bandwidth"]  # bytes/s

    # 稀疏算子参数
    nnz = sparse_op["nnz"]

    ridge_point = peak_compute / peak_bandwidth

    if compute_intensity < ridge_point:
        # 内存受限
        achievable_perf = compute_intensity * peak_bandwidth
        bottleneck = "memory"
    else:
        # 计算受限
        achievable_perf = peak_compute
        bottleneck = "compute"

    return {
        "achievable_performance": achievable_perf,
        "bottleneck": bottleneck,
        "compute_intensity": compute_intensity,
        "efficiency": achievable_perf / peak_compute,
    }
```

```python
def analyze_sparse_properties(matrix):
    """分析稀疏矩阵的性质"""

    properties = {}

    # 1. 稀疏度
    nnz = np.count_nonzero(matrix)
    total = matrix.size
    properties["sparsity"] = 1.0 - nnz / total

    # 2. 行非零元素分布
    row_nnz = np.count_nonzero(matrix, axis=1)
    properties["row_nnz_mean"] = np.mean(row_nnz)
    properties["row_nnz_std"] = np.std(row_nnz)

    properties["col_nnz_std"] = np.std(col_nnz)

    bandwidth = 0
    for i in range(matrix.shape[0]):
        nonzero_cols = np.nonzero(matrix[i])[0]
        if len(nonzero_cols) > 0:

                          nonzero_cols[-1] - nonzero_cols[0])

    properties["is_block_sparse"] = detect_block_structure(matrix)

    diag = np.abs(np.diag(matrix))
```

```c
void parallel_spmm_csr(
    const float* A_data,
    const int* A_indices,
    const int* A_indptr,
    const float* B,
    float* C,
    int m, int n, int num_threads
) {
    #pragma omp parallel num_threads(num_threads)
    {
        int tid = omp_get_thread_num();
        int rows_per_thread = m / num_threads;
        int start_row = tid * rows_per_thread;
        int end_row = (tid == num_threads - 1) ? m : start_row + rows_per_thread;

        for (int i = start_row; i < end_row; i++) {
            for (int j = 0; j < n; j++) {
                C[i * n + j] = 0.0f;
            }
            for (int p = A_indptr[i]; p < A_indptr[i+1]; p++) {
                int col = A_indices[p];
                float val = A_data[p];
                for (int j = 0; j < n; j++) {
                    C[i * n + j] += val * B[col * n + j];
                }
            }
        }
    }
}
```

```cuda
__global__ void spmv_csr_warp_kernel(
    const float* data,
    const int* indices,
    const int* indptr,
    const float* x,
    float* y,
    int num_rows
) {
    int row = blockIdx.x;
    int lane = threadIdx.x;

    if (row < num_rows) {
        int start = indptr[row];
        int end = indptr[row+1];
        float sum = 0.0f;

        for (int j = start + lane; j < end; j += 32) {
            sum += data[j] * x[indices[j]];

        for (int offset = 16; offset > 0; offset /= 2) {
            sum += __shfl_down_sync(0xffffffff, sum, offset);
        }

        if (lane == 0) {
            y[row] = sum;
        }
    }
}
```

```c
void cache_friendly_spmv(
    const float* data,
    const int* indices,
    const int* indptr,
    const float* x,
    float* y,
    int num_rows,
    int cache_line_size
) {
    // 分块处理，每块适配缓存行
    int block_size = cache_line_size / sizeof(float);

    for (int i = 0; i < num_rows; i++) {
        float sum = 0.0f;
        int start = indptr[i];
        int end = indptr[i + 1];

        // 预取下一个缓存行
        if (end - start > block_size) {
            __builtin_prefetch(&data[start + block_size], 0, 1);
            __builtin_prefetch(&indices[start + block_size], 0, 1);
        }

        for (int j = start; j < end; j++) {
            sum += data[j] * x[indices[j]];
        }

        y[i] = sum;
    }
}
```

```c
void avx2_spmm(
    const float* A_data,
    const int* A_indices,
    const int* A_indptr,
    const float* B,
    float* C,
    int m, int n
) {
    for (int i = 0; i < m; i++) {
        // 初始化输出行为零
        __m256 zero = _mm256_setzero_ps();
        for (int j = 0; j < n; j += 8) {
            _mm256_store_ps(&C[i * n + j], zero);
        }

        // 累加非零元素
        for (int p = A_indptr[i]; p < A_indptr[i + 1]; p++) {
            int col = A_indices[p];
            float val = A_data[p];
            __m256 vval = _mm256_set1_ps(val);

                __m256 vb = _mm256_load_ps(&B[col * n + j]);
                __m256 vc = _mm256_load_ps(&C[i * n + j]);
                vc = _mm256_fmadd_ps(vval, vb, vc);
                _mm256_store_ps(&C[i * n + j], vc);
            }
        }
    }
```

```cpp
// 文件: include/tvm/runtime/ndarray.h
// TVM 中稀疏 NDArray 的内部表示

// TVM 使用 CSR 格式存储稀疏张量
struct CSRNDArray {
  NDArray data;      // 非零元素值 (shape: [nnz])
  NDArray indices;   // 列索引 (shape: [nnz], dtype: int32)
  NDArray indptr;    // 行指针 (shape: [num_rows+1], dtype: int32)

  // 从稠密张量转换
  static CSRNDArray FromDense(const NDArray& dense) {
    auto shape = dense.Shape();
    int num_rows = shape[0];
    int num_cols = shape[1];

    // 第一遍: 计算每行非零元素数
    std::vector<int> row_counts(num_rows, 0);
    auto* dense_data = static_cast<float*>(dense->data);
    for (int i = 0; i < num_rows; i++) {
      for (int j = 0; j < num_cols; j++) {
        if (dense_data[i * num_cols + j] != 0.0f) {
          row_counts[i]++;
        }
      }
    }

    // 计算总非零元素数
    int nnz = std::accumulate(row_counts.begin(), row_counts.end(), 0);

    // 第二遍: 填充 CSR 数组
    CSRNDArray csr;
    csr.data = NDArray::Empty({nnz}, DataType::Float(32), CPU);
    csr.indices = NDArray::Empty({nnz}, DataType::Int(32), CPU);
    csr.indptr = NDArray::Empty({num_rows + 1}, DataType::Int(32), CPU);

    auto* values = static_cast<float*>(csr.data->data);
    auto* cols = static_cast<int*>(csr.indices->data);
    auto* row_ptr = static_cast<int*>(csr.indptr->data);

    row_ptr[0] = 0;
    for (int i = 0; i < num_rows; i++) {
      row_ptr[i + 1] = row_ptr[i] + row_counts[i];
    }

    int idx = 0;
    for (int i = 0; i < num_rows; i++) {
      for (int j = 0; j < num_cols; j++) {
        float val = dense_data[i * num_cols + j];
        if (val != 0.0f) {
          values[idx] = val;
          cols[idx] = j;
          idx++;
        }
      }
    }

    return csr;
  }
};
```

```python
def register_sparse_operators():
    """在 TVM 中注册稀疏算子"""

    # CSR-SpMV 算子注册
    @tvm.register_func("tvm.sparse.csr_spmv")
    def csr_spmv(values, indices, indptr, x, y):
        """CSR 格式的稀疏矩阵-向量乘法

        参数:
            values: CSR 非零元素 (1D tensor)
            indices: CSR 列索引 (1D int32 tensor)
            indptr: CSR 行指针 (1D int32 tensor)
            x: 输入向量 (1D tensor)
            y: 输出向量 (1D tensor)
        """
        num_rows = indptr.shape[0] - 1

        # 使用 TIR 定义计算
        i = te.axis.spatial(num_rows, name="i")
        k = te.reduce_axis(
            (te.min(indptr[i], indptr[i+1]),
             te.max(indptr[i], indptr[i+1])),
            name="k"
        )

        # CSR-SpMV 计算
        y[i] = te.sum(
            values[k] * x[indices[k]],
            axis=k
        )

        return te.compute(
            (num_rows,),
            lambda i: y[i],
            name="csr_spmv_output"
        )

    # CSR-SpMM 算子注册
    @tvm.register_func("tvm.sparse.csr_spmm")
    def csr_spmm(values, indices, indptr, B, C, n):
        """CSR 格式的稀疏矩阵-稠密矩阵乘法"""
        num_rows = indptr.shape[0] - 1

        i = te.axis.spatial(num_rows, name="i")
        j = te.axis.spatial(n, name="j")
        k = te.reduce_axis(
            (0, indptr[i+1] - indptr[i]),
            name="k"
        )

        C[i, j] = te.sum(
            values[indptr[i] + k] * B[indices[indptr[i] + k], j],
            axis=k
        )

        return C
```

```python
def sparse_attention_forward(Q, K, V, attention_mask):
    """稀疏注意力机制

    标准注意力: O(n²) 复杂度
    稀疏注意力: O(n * k) 复杂度，k << n

    常见稀疏模式:
    1. 局部窗口注意力（Local Window）
    2. 全局注意力（Global Tokens）
    3. 随机注意力（Random Attention）
    4. 扩张注意力（Dilated Attention）
    """

    n = Q.shape[0]  # 序列长度
    d = Q.shape[1]  # 特征维度

    # 构建稀疏注意力掩码
    sparse_mask = build_sparse_mask(n, pattern="local+global")

    # 将掩码转换为 CSR 格式
    rows, cols = np.where(sparse_mask)
    values = np.ones(len(rows), dtype=np.float32)

    # CSR 格式
    indptr = np.zeros(n + 1, dtype=np.int32)
    for i in range(len(rows)):
        indptr[rows[i] + 1] += 1
    indptr = np.cumsum(indices=indptr)

    # 稀疏 QK^T 计算
    # 只计算掩码中为 1 的位置
    attention_scores = np.zeros((n, n), dtype=np.float32)
    for i in range(n):
        start, end = indptr[i], indptr[i + 1]
        for idx in range(start, end):
            j = cols[idx]
            attention_scores[i, j] = np.dot(Q[i], K[j]) / np.sqrt(d)

    # 应用 softmax（只在非零位置）
    attention_probs = sparse_softmax(attention_scores, indptr, cols)

    # 稀疏 PV 乘法
    output = np.zeros((n, d), dtype=np.float32)
    for i in range(n):
        start, end = indptr[i], indptr[i + 1]
        for idx in range(start, end):
            j = cols[idx]
            output[i] += attention_probs[i, j] * V[j]

    return output


def build_sparse_mask(n, pattern="local+global"):
    """构建稀疏注意力掩码"""
    mask = np.zeros((n, n), dtype=bool)
    window_size = 256  # 局部窗口大小
    global_tokens = 16  # 全局 token 数量

    if "local" in pattern:
        # 局部窗口注意力
        for i in range(n):
            start = max(0, i - window_size // 2)
            end = min(n, i + window_size // 2)
            mask[i, start:end] = True

    if "global" in pattern:
        # 前 global_tokens 个 token 可以关注所有位置
        mask[:global_tokens, :] = True
        mask[:, :global_tokens] = True

    return mask
```

```python
def sparse_embedding_lookup(embedding_table, indices):
    """稀疏 Embedding 查找

    推荐系统中，Embedding 表非常大（百万级 item），
    但每次只查找少量 item（batch_size << num_items）。

    这本质上是一个稀疏操作。

    参数:
        embedding_table: 稀疏存储的 Embedding 表 (CSR 格式)
        indices: 要查找的 item 索引 (batch_size,)
    """

    batch_size = len(indices)
    embed_dim = embedding_table.shape[1]

    # 输出缓冲区
    output = np.zeros((batch_size, embed_dim), dtype=np.float32)

    # 批量查找
    for i, idx in enumerate(indices):
        # 从 CSR 格式中提取第 idx 行
        start = embedding_table.indptr[idx]
        end = embedding_table.indptr[idx + 1]

        if start < end:
            # 非稀疏行：直接复制
            cols = embedding_table.indices[start:end]
            vals = embedding_table.data[start:end]
            output[i, cols] = vals
        # 稀疏行：保持零值

    return output

    # 计算非零特征的位置
    nonzero_mask = sparse_features != 0
    nonzero_counts = nonzero_mask.sum(axis=1)

    # 稀疏外积（只计算非零位置的交互）
    interactions = []
    for i in range(len(sparse_features)):
        nz_idx = np.where(nonzero_mask[i])[0]
        if len(nz_idx) > 1:
            # 只对非零特征做外积
            nz_vals = sparse_features[i, nz_idx]
            outer = np.outer(nz_vals, nz_vals)
            interactions.append((nz_idx, outer))

    return interactions
```

```python
def run_sparse_test_suite():
    """稀疏计算完整测试套件"""

    import scipy.sparse as sp

    test_cases = [
        # (name, matrix_size, sparsity, format)
        ("小矩阵-中等稀疏", (100, 100), 0.9, "csr"),
        ("小矩阵-高度稀疏", (100, 100), 0.99, "csr"),
        ("大矩阵-中等稀疏", (10000, 10000), 0.9, "csr"),
        ("大矩阵-极度稀疏", (10000, 10000), 0.999, "csr"),
        ("长方形矩阵", (1000, 5000), 0.95, "csr"),
        ("COO 格式", (1000, 1000), 0.95, "coo"),
        ("BSR 格式", (1024, 1024), 0.9, "bsr"),
    ]

    results = []
    for name, shape, sparsity, fmt in test_cases:
        density = 1 - sparsity
        A = sp.random(shape[0], shape[1], density=density, format=fmt)

        y_dense = A.toarray() @ x
        y_sparse = A @ x
        correct = np.allclose(y_dense, y_sparse, atol=1e-5)

        start = time.time()
        for _ in range(100):

        elapsed = (time.time() - start) / 100

        results.append({
            "name": name,
            "shape": shape,
            "sparsity": sparsity,
            "format": fmt,
            "correct": correct,
            "time_ms": elapsed * 1000,
        })

    # 打印结果
    print("测试名称              | 形状         | 稀疏度 | 格式 | 正确 | 时间(ms)")
    print("-" * 80)
    for r in results:
        status = "PASS" if r["correct"] else "FAIL"
        print(f"{r['name']:22s}| {str(r['shape']):12s} | "
              f"{r['sparsity']:.3f}  | {r['format']:4s} | {status:4s} | "
              f"{r['time_ms']:.3f}")

    return results
```
