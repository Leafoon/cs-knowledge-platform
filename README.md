# CS Knowledge Platform

专业级计算机知识学习平台

## 技术栈

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: Markdown
- **Animations**: Framer Motion

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建静态站点
npm run build
```

## 项目结构

```
app/              # Next.js App Router 页面
components/       # React 组件
  ├── layout/     # 布局组件
  ├── knowledge/  # 知识模块组件
  └── ui/         # 基础 UI 组件
content/          # Markdown 内容
lib/              # 工具函数
types/            # TypeScript 类型定义
public/           # 静态资源
styles/           # 样式文件
```
